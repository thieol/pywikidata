= what is it ? =

pywikidata can be use to :
- read/modify labels
- read/modify descriptions
- read/write aliases

= limitations =

it has been tested only on python 2.7
and only for labels and descriptions

= quickstart =
edit config.properties and put your own settings
put config.properties in the main path of your program
pywikidata will then search for a file named config.properties
now see examples.py to see how you can use wikidata
